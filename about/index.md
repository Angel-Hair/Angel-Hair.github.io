# 关于"Angel Hair"

我在网络上的一般性名字（然而用这个名字的人并不一定就是我，腚!），小时候看过一个自然科学文章后以此作为自己的网名之后再也没有改变过。
渐渐的，我的友人们都开始用缩略语——“AH”来称呼我了，所以抱着 `啊，就这样吧_(:зゝ∠)_` 的想法，决定让这个名字就这样永远伴随着自己好了~~其实是懒得想名字~~ 。
